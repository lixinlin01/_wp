# JavaScript 函數實作練習題

## 題目列表

| # | 題目 | 說明 |
|---|------|------|
| 1 | Callback 基礎實作 | 實作 `mathTool(num1, num2, action)`，透過 callback 執行加減運算 |
| 2 | IIFE | 立即執行函數，封裝區域變數 `count` |
| 3 | 箭頭函數與 map | 使用箭頭函數搭配 `map` 將價格打 8 折 |
| 4 | 破壞性修改 | 實作 `cleanData(arr)`，探討陣列的傳址特性 |
| 5 | Higher-Order Function | 實作 `multiplier(factor)` 回傳新函數 |
| 6 | Callback 篩選器 | 手寫 `myFilter(arr, callback)` 模擬 `filter` |
| 7 | 箭頭與物件篩選 | 使用 `filter` + 箭頭函數篩選成年人 |
| 8 | 參數傳址陷阱 | 比較 `a.push()` 與 `b = [...]` 的差異 |
| 9 | 延遲 Callback | `setTimeout` + 箭頭函數組合字串 |
| 10 | 綜合應用 | 購物車總價計算 + callback 折扣 |

## 使用方式

執行任一練習：

```bash
node 01_callback_basic.js
node 10_calculate_total.js
```
