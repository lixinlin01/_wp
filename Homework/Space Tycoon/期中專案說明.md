## 遊戲說明
**這是一個基於 Web 雙向互動的星際主題模擬交易遊戲。玩家將扮演一名穿梭於星系的獨立交易員，在為期 30 天的交易期內，透過買低賣高與應對突發事件來最大化總資產，並挑戰全星系財富排行榜。**

---

## 🎮 遊戲玩法

1. **建立身分**：在首頁輸入「交易員代號」進行身分驗證，系統會為你建立專屬的交易帳戶並配發初始資金 $5,000。
2. **觀察行情**：進入交易網後，實時星際行情表會顯示各項物資的「現價」與「漲跌趨勢」。點擊下方的星系資源產地，可查看該星球的詳細情報與風險評估。
3. **執行交易**：在操作面板輸入欲交易的「數量」，點擊「買入」或「賣出」。系統會自動判斷資金與庫存，完成扣款或入帳，並即時更新貨艙清單。
4. **推進天數**：完成當日操作後，點擊右上角的「推進天數 (NEXT DAY)」。市場價格將根據各物資的波動率重新計算，同時星際情報網可能播報突發新聞，引發特定物資價格的劇烈震盪。
5. **結算與排行**：當時間推進至第 30 天，系統將強制結算剩餘現金與庫存現值。最終的「總資產」將上傳至伺服器，並展示於首頁的排行榜中與全星系玩家一較高下。

## 📦 遊戲物件說明

遊戲中共有四種核心交易物資，分別對應不同的產地星球與投資風險。風險越高的物資，每日價格波動的幅度越大。

| 物資名稱 | 產地星球 | 風險等級 | 波動率 | 物件特性與市場定位 |
| :--- | :--- | :--- | :--- | :--- |
| **太空口糧** | 農業星 Demeter | 極低 (安全避險) | 10% | 溫室覆蓋率達 90% 的農業行星。投資回報率低，但在市場動盪、高風險物資崩盤時，是保本的最佳避風港。 |
| **鈦合金** | 礦業星 Titania | 中等 (工業週期) | 25% | 重工業星球產物。價格穩定，通常跟隨全星系的星艦製造成本與軍工政策起伏。 |
| **量子晶片** | 科技星 Silicon-9 | 高 (技術突破) | 50% | 晶片良率與新技術發表會導致價格劇烈震盪，是投機客與追求高報酬者的最愛。 |
| **反物質** | 邊界星 Void-X | 極高 (致命危險) | 80% | 位於黑洞邊緣提煉的極危險物質。可能發生提煉廠爆炸等極端事件，能讓人一夜暴富或瞬間破產。 |

## 🏗️ 系統架構說明

本專案採用前後端分離的基礎架構，未使用重量級框架，專注於原生技術的邏輯處理與 API 串接。

* **前端 (Frontend)**
  * **技術棧**：HTML5, CSS3, Vanilla JavaScript。
  * **狀態管理**：使用瀏覽器 `localStorage` 記錄玩家的 Session ID，確保重新整理網頁時遊戲進度不會遺失。
  * **非同步通訊**：透過原生 `fetch` API 與後端進行 JSON 格式的資料交換。
* **後端 (Backend)**
  * **技術棧**：Node.js, Express.js。
  * **路由與邏輯**：負責處理 RESTful API 請求，包含身分註冊 (`/api/start`)、交易驗證 (`/api/trade`)、天數推進與隨機事件引擎 (`/api/next-day`)。
* **資料庫 (Database)**
  * **技術棧**：SQLite3 (`tycoon.db`)。
  * **資料表結構**：持久化儲存全球排行榜與玩家當前遊戲狀態，確保不同玩家連線時的資料隔離。
