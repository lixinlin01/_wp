## 習題 4 — JavaScript 10 題練習

**路徑：** [`homework/week4hw/`](../homework/week4hw)

10 個獨立 JS 程式，涵蓋基礎語法與邏輯：

| 練習編號 | 主題目標 | 涵蓋觀念 | 核心邏輯簡述 |
| :--- | :--- | :--- | :--- |
| **練習 1** | 計算陣列總和 | Function, Array, For | 宣告總和變數為 0，使用 `for` 迴圈走訪陣列，將所有數字逐一累加。 |
| **練習 2** | 判斷物件屬性與條件 | Function, Object, If | 讀取物件內的 `age` 屬性，使用 `if...else` 判斷數值是否大於等於 18 並回傳對應字串。 |
| **練習 3** | 尋找特定數字 | Function, Array, While, If | 使用 `while` 迴圈走訪陣列，搭配 `if` 條件比對元素，找到目標數字即中斷並回傳索引值。 |
| **練習 4** | 解析 JSON 字串提取資料 | Function, JSON, Array, Object, For | 以 `JSON.parse` 將字串轉為陣列物件，用 `for` 迴圈提取物件的 `name` 屬性並存入新陣列。 |
| **練習 5** | 尋找最高分學生 | Function, Array, Object, For, If | 假設首筆資料為最高分基準，利用 `for` 與 `if` 逐一比較後續物件的分數屬性並動態更新最高分。 |
| **練習 6** | 統計陣列元素出現次數 | Function, Array, Object, For, If | 用 `for` 走訪陣列，並以 `if` 檢查空物件中是否已有該元素作為屬性，若有則累加，無則初始化為 1。 |
| **練習 7** | 購物車結帳系統 | Function, Array, Object, While | 設定索引值，使用 `while` 迴圈逐一取得陣列中物件，計算「數量 × 單價」並持續加總。 |
| **練習 8** | 資料篩選與轉換 JSON | Function, JSON, Array, Object, For, If | 用 `for` 與 `if` 篩選庫存大於 0 的商品加入新陣列，最後以 `JSON.stringify` 轉回 JSON 字串。 |
| **練習 9** | 字串反轉器 | Function, String(Array), While | 將字串視為陣列，設定索引為字串長度減 1，用 `while` 迴圈從尾端倒序取字元並拼湊成新字串。 |
| **練習 10** | 綜合資料清洗與分類 | Function, JSON, Array, Object, While, For, If | 結合所有觀念：解析 JSON，用 `while` 剔除取消的訂單，再用 `for` 依 VIP 條件將物件推入分類物件的不同陣列中。 |

## 習題 5 — 電商網站專案（三版本）

**路徑：** [`ecommerce-website/`](../ecommerce-website)

使用 React + Vite 建置的電商網站，分為三個版本：

### v1-basic（簡潔版）
- 技術：`useState` + 單頁式設計
- 功能：商品列表、側邊購物車、加減數量

### v2-modern（現代版）
- 技術：`Context API` + `React Router`
- 功能：首頁 Hero、分類篩選、搜尋、購物車頁面
- 設計：紫藍漸層主題

### v3-enterprise（專業版）
- 技術：`Zustand` + `React Router` + 完整路由
- 功能：商品詳情、結帳表單驗證、訂單確認、免運機制
- 設計：深色主題 + 紅金強調色
